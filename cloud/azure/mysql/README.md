# CLOUD AZURE MYSQL DataDog monitors

## How to use this module

```hcl
module "datadog-monitors-cloud-azure-mysql" {
  source      = "claranet/monitors/datadog//cloud/azure/mysql"
  version     = "{revision}"

  environment = var.environment
  message     = module.datadog-message-alerting.alerting-message
}

```

## Purpose

Creates DataDog monitors with the following checks:

- Mysql ${title(var.server_type)} Server CPU usage
- Mysql ${title(var.server_type)} Server IO consumption
- Mysql ${title(var.server_type)} Server memory usage
- Mysql ${title(var.server_type)} Server storage
